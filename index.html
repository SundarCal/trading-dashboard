How to export from IBKR (Interactive Brokers)

  1. Log in to Client Portal at clientportal.ibkr.com
  2. Click Performance & Reports in the top menu
  3. Select Flex Queries from the dropdown
  4. In "Activity Flex Queries" click the ＋ plus icon to create a new query
  5. Name it (e.g. "My Trades"), then under Sections select Trades → Select All → Save
  6. Back on the Flex Queries page, click the ▶ Run button next to your query
  7. Choose your date range (max 365 days per export), select CSV format, click Run
  8. Download the CSV file and upload it here
⚠ IBKR limits each export to 365 days. For multi-year history, run the export multiple times with different date ranges and import each file separately.

How to export from Fidelity

  1. Log in to Fidelity.com
  2. Go to Accounts & Trade → History from the top navigation
  3. Select your account from the left panel
  4. Set your Date Range (up to 90 days per download) and click Apply
  5. Click the Download icon near the top-right of the activity table
  6. Select CSV as the format and save the file
  7. Upload it here — the file is usually named History_for_Account_ending_in_XXXX.csv
⚠ Fidelity limits each download to 90 days. For a full year you'll need 4–5 separate downloads — import each one here and they'll be merged together.

How to export from Charles Schwab

  1. Log in to schwab.com
  2. Click the Accounts tab in the top navigation
  3. Select History from the dropdown submenu
  4. If you have multiple accounts, pick the correct account from the dropdown
  5. Set the filter to All Transactions (click "Filter by Transaction Types → Select All → Apply")
  6. Choose your date range (Custom or preset), then click Search
  7. Click the Export button in the top-right corner
  8. Select CSV as the export type and click Export again
  9. Save the file and upload it here
⚠ Schwab limits exports to 1,500 records and no option trade times. For large histories, download by quarter. If you use thinkorswim, you can also export from there for better options data.
